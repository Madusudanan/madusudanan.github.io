This page lists all the developer tools that I have come across that are really good at what they do.Of course, there might be better ones.Do drop me a note if you find any.


Tool Name Description Link
Epic Editor

User for : Markdown editing

If you ever wanted to preview markdown and how it wanted to look like, epic editor can be a pretty handy one.No installation required, head over to the website, the tool is entirely developed in Javascript and hence client side and can be used offline.
Git

User for : Version Control

When it comes to Version control, Git is probably most widely used and one of the best.It has many cloud providers as well. See this video for a not gentle introduction to Git.
Trello

Used for : Task management

There are many tools for task management, but trello is what I found as the one that stands out for personal usage.Completely free, web based and there are also pretty good mobile app counterparts.
Webstorm

Used for : Web development

What ?? An IDE for web development ? Yes, I used to love brackets until I found webstorm, this is pretty kickass and has a ton of features and a slick UI.Start using it and you will fall in love with it.

P.S - It is commercial, but you can get a free license for open source projects.
